# 編碼與解碼

## 數字信號常用編碼方法

![[digital_signal_encoding_methods.png]]

- 不归零编码 NRZ
  - 编码规则
    - 低电平 -> 0
    - 高电平 -> 1
  - 没有检错功能
  - 没有自同步功能
- 归零编码 RZ
  - 编码规则
    - 信号周期初始低电平 -> 0
    - 信号周期初始高电平 -> 1
    - 信号周期中间必须归零
  - 自同步
- 反向不归零编码 NRZI
  - 编码规则
    - 信号周期初始跳变 -> 0
    - 信号周期初始保持 -> 1
  - 没有自同步功能
- 曼切斯特编码
  - 编码规则
    - 中间上跳 -> 0
    - 中间下跳 -> 1
  - 自同步
- 差分曼切斯特编码
  - 编码规则
    - 中间跳变
    - 信号周期初始跳变 -> 0
    - 信号周期初始保持 -> 1
  - 自同步
- 4B/5B 编码

## 脈衝編碼調製

- 采样
  - 周期性扫描模拟信号，把时间上连续的信号变成离散的信号
  - 奈奎斯特-香農采樣定理: 采样频率 ≥ 2 × 信号最高频率
- 量化
  - 把采样取得的电平按照一定的分级标度转化为对应的数值，并取整数
- 编码
  - 把量化的结果转换为与之对应的二进制编码
