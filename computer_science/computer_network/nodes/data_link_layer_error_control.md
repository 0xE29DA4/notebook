## 檢錯與糾錯

### 僅檢錯校驗碼

#### 奇偶校验码

检错能力

- 奇数位错误

#### CRC 循环冗余校验码

- 约定好一个生成多项式
- 校验位位数 R: 生成多项式最高次幂
- 被除数：信息位 + R 个 0
- 除数：由生成多项式确定
- 校验码：余数

检错能力:

- 所有奇数位错误
- 所有双比特错误
- 所有长度小于等于 R 的突发错误

### 可糾錯校驗碼

#### 海明碼

- 信息 n 位，校验 k 位，
- 要求：2^k >= n + k + 1
- 设信息位 Dn...D2D1
- 设校验位 Pk...P2P1
- 对应海明码 Hn+k...H2H1
- 校验位 Pi 放在海明码 2^(i-1) 的位置上（放在 H1, H2, H4, H8...）

检错能力:

- 2 位
- 为了区分 1 位错还是 2 位错，通常还需添加 1 位全校验位

纠错能力:

- 1 位
