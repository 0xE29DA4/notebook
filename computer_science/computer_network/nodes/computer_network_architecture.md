# 計算機網路體系結構

> 计算机网络体系结构是计算机网络各层及其协议的集合，就是这个计算机网络及其构件所应完成的功能的精确定义。实现(implementation)是遵循这种体系结构的前提下，用何种硬件或软件完成这些功能的问题。体系结构是抽象的，而实现则是具体的。

- 实体
  - 计算机网络分层结构中，第 n 层活动元素称为称为第 n 层实体
- 对等层
  - 不同机器上的同一层称为对等层
- 对等实体
  - 同一层的实体称为对等实体
- 协议
  - 是控制对等实体之间进行通信的规则的集合
  - 是水平的
  - 三要素
    - 语法
    - 语义
    - 时序
- 服务
  - 指下层为紧邻的上层提供的功能调用
它是垂直的
- 接口
  - 同一节点相邻两层实体交换信息的逻辑接口，又称为服务访问点（Service Access Point，SAP）
- 协议数据单元 PDU
  - 对等实体之间传送数据的单位
  - 服务数据单元 SDU
    - 为完成上一层实体所要求的功能而传送的数据
  - 协议控制信息 PCI
    - 控制协议操作的信息
