# Network Layer

- Task: 实现主机到主机通信
- Functions
  - 异构网络互联：不同网络可能拓扑结构不同，链路层、物理层实现不同，网络层屏蔽了这些底层差异
  - 路由选择
    - 路由器运行路由协议，最终生成各自的路由表
    - 路由器之间相互配合，规划分组的最佳转发路径
  - 分组转发
    - 路由器根据转发表，将收到的分组从合适的端口转发出去
  - 拥塞控制
    - 拥塞：网络上出现过量分组，引起网络性能下降
    - 拥塞控制方法
      - 开环控制
      - 闭环控制
  - 其他功能
    - 差错控制
    - 流量控制
    - 连接建立与释放
    - 可靠传输管理
