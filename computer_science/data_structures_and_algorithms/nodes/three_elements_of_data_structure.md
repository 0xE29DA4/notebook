# 數據結構三要素

- 邏輯結構
  - 線性結構
    - 數據元素之間是一對一的線性關係
  - 非線性結構
    - 集合結構：數據元素之間沒有關係
    - 樹形結構：數據元素之間是一對多的關係
    - 圖形結構：數據元素之間是多對多的關係
- 物理結構
  - 順序存儲結構
    - 把邏輯上相鄰的元素存儲在物理位置上也相鄰的存儲單元中，元素之間的關係由存儲單元的鄰接關係來體現
  - 非順序存儲結構
    - 鏈式存儲結構：用一組任意的存儲單元存儲數據元素，數據元素之間的邏輯關係用指針來表示
    - 索引存儲結構：存儲元素信息的同時，還建立附加的索引表
    - 散列存儲結構：又稱 Hash 存儲，根據元素的關鍵字直接計算出該元素的存儲地址
- 運算
  - 施加在數據上的運算，包括運算的定義和實現
  - 運算的定義是針對邏輯結構的，指出運算的功能
  - 運算的實現是針對存儲結構的，指出運算的具體操作步驟
