# 數據結構基本概念

- 数据
  - 数值型：整数、实数
  - 非数值型：图像、音频、文字等
- 数据元素：数据的基本单位
- 数据项
  - 构成数据元素的不可分割的最小单位
  - 如果一个数据项由多个细分的属性组成，这个数据项称为组合项
- 数据结构：存在一种或多种特定关系的数据元素的集合
- 数据对象：具有相同性质的数据元素的集合，是数据的一个子集
- 数据类型：一个值的集合和定义在此集合上的一组操作的总称
  - 分类
    - 原子类型：其值不可再分
    - 结构类型：其值可以再分解为若干分量
- ADT, 抽象数据类型
  - ADT = 逻辑结构 + 抽象运算
  - 不考虑存储结构和其数据项，只考虑逻辑结构和基本操作
