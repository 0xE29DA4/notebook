# age

> 一个简单、现代、安全的文件加密工具。
> 更多信息：<https://age-encryption.org>

- 使用公钥加密文件：
  `age -r {{公钥}} -o {{加密文件.age}} {{源文件}}`

- 使用密码加密文件：
  `age -p -o {{加密文件.age}} {{源文件}}`

- 使用密钥文件加密：
  `age -i {{密钥文件}} -o {{加密文件.age}} {{源文件}}`

- 解密文件：
  `age -d -i {{密钥文件}} {{加密文件.age}} > {{解密文件}}`

- 使用密码解密文件：
  `age -d {{加密文件.age}} > {{解密文件}}`

- 加密到多个接收者：
  `age -r {{公钥1}} -r {{公钥2}} -o {{加密文件.age}} {{源文件}}`

- 查看加密文件的元信息（不解密）：
  `age --decrypt --list {{加密文件.age}}`
