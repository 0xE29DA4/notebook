# cat

> 连接并显示文件内容。
> 更多信息：<https://manned.org/cat>

- 显示文件内容：
  `cat {{文件}}`

- 显示多个文件内容：
  `cat {{文件1}} {{文件2}}`

- 连接多个文件到目标文件：
  `cat {{文件1}} {{文件2}} > {{目标文件}}`

- 追加文件内容到另一个文件：
  `cat {{文件}} >> {{目标文件}}`

- 显示行号：
  `cat -n {{文件}}`

- 显示行号（空行除外）：
  `cat -b {{文件}}`

- 显示行尾结束符（$）：
  `cat -E {{文件}}`

- 显示制表符（^I）：
  `cat -T {{文件}}`

- 显示不可见字符：
  `cat -A {{文件}}`
