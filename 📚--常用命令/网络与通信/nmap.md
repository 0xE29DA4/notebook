# nmap

> 网络扫描和安全审计工具。
> 更多信息：<https://manned.org/nmap>

- 扫描主机是否在线（不进行端口扫描）：
  `nmap -sn {{192.168.1.0/24}}`

- 扫描指定主机的常用 1000 个端口：
  `nmap {{主机名}}`

- 扫描所有 65535 个端口：
  `nmap -p- {{主机名}}`

- 扫描指定端口：
  `nmap -p {{22,80,443}} {{主机名}}`

- 扫描并识别服务和操作系统：
  `nmap -A {{主机名}}`

- 快速扫描（只扫描常用端口）：
  `nmap -F {{主机名}}`

- 使用 TCP SYN 扫描（隐蔽扫描，需要 root）：
  `sudo nmap -sS {{主机名}}`

- 使用 UDP 扫描：
  `sudo nmap -sU {{主机名}}`

- 扫描结果保存到文件：
  `nmap -oN {{output.txt}} {{主机名}}`
