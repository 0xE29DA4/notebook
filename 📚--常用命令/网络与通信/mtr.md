# mtr

> 网络诊断工具（结合 ping 和 traceroute 功能）。
> 更多信息：<https://manned.org/mtr>

- 实时追踪到主机的路由：
  `mtr {{主机名}}`

- 以报告模式运行（发送 10 个包后退出）：
  `mtr -r {{主机名}}`

- 以报告模式运行，发送指定数量的包：
  `mtr -r -c {{数量}} {{主机名}}`

- 显示 IP 地址而非主机名：
  `mtr -n {{主机名}}`

- 指定发送数据包的间隔（秒）：
  `mtr -i {{秒数}} {{主机名}}`

- 使用 TCP SYN 进行探测：
  `mtr -T {{主机名}}`

- 使用 UDP 数据报进行探测：
  `mtr -u {{主机名}}`

- 显示扩展信息（AS 号、IP、丢包率等）：
  `mtr -e {{主机名}}`
