# rsync

> 快速、多功能的文件复制和同步工具。
> 更多信息：<https://manned.org/rsync>

- 同步本地目录（归档模式）：
  `rsync -av {{源目录/}} {{目标目录/}}`

- 通过 SSH 同步到远程服务器：
  `rsync -avz {{源目录/}} {{用户@主机:目标目录/}}`

- 通过 SSH 从远程服务器同步：
  `rsync -avz {{用户@主机:源目录/}} {{目标目录/}}`

- 同步并删除目标端多余文件：
  `rsync -avz --delete {{源目录/}} {{目标目录/}}`

- 显示同步进度：
  `rsync -avz --progress {{源目录/}} {{目标目录/}}`

- 排除指定文件：
  `rsync -av --exclude "{{*.tmp}}" {{源目录/}} {{目标目录/}}`

- 只更新较新的文件：
  `rsync -avu {{源目录/}} {{目标目录/}}`

- 使用压缩传输：
  `rsync -avz {{源目录/}} {{目标目录/}}`
