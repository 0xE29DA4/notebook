# Data Type

## Numeric

Integer

- SMALLINT
- INTEGER(INT is a abbreviation for it)
- BIGINT
- SERIAL
- SMALLSERIAL
- BIGSERIAL

Floating Point

- REAL
- DOUBLE PRECISION(DOUBLE is a abbreviation for it)
- FLOAT(n)
- DECIMAL(n)/NUMERIC(n)

## String and Sequence

- CHAR(n)
- VARCHAR(n)
- TEXT

> Support array type. For example: `INTEGER[]`, `TEXT[]`, etc.

## Date and Time

- DATE
- TIME
- TIMETZ
- TIMESTAMP
- TIMESTAMPTZ
- INTERVAL

## Blob

- BYTEA

## Other

- BOOLEAN
- UUID
- JSON
- JSONB
- ARRAY
- INET
- CIDR
