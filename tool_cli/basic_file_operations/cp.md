# cp

> 复制文件和目录。
> 更多信息：<https://manned.org/cp>

- 将文件复制到新位置：
  `cp {{路径/到/源文件}} {{路径/到/目标文件}}`

- 将文件复制到另一个目录，保留原文件名：
  `cp {{路径/到/源文件}} {{路径/到/目标目录}}`

- 递归复制目录及其内容到新位置（如果目标目录已存在，则复制到其内部）：
  `cp -r {{路径/到/源目录}} {{路径/到/目标目录}}`

- 在覆盖现有文件前进行提示：
  `cp -i {{路径/到/源文件}} {{路径/到/目标目录}}`

- 复制的同时保持文件原始属性（如权限、所有者、时间戳）：
  `cp -p {{路径/到/源文件}} {{路径/到/目标文件}}`
