# sort

> 对文本文件的行进行排序。
> 更多信息：<https://manned.org/sort>

- 按字典序排序：
  `sort {{文件}}`

- 按数值排序：
  `sort -n {{文件}}`

- 逆序排序：
  `sort -r {{文件}}`

- 去重排序：
  `sort -u {{文件}}`

- 按指定列排序（如第 2 列）：
  `sort -k 2 {{文件}}`

- 按数值逆序排序：
  `sort -nr {{文件}}`

- 按人类可读的数值排序（如 1K, 2M）：
  `sort -h {{文件}}`

- 排序并输出到文件：
  `sort {{文件}} -o {{输出文件}}`
