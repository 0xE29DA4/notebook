# test

> 条件测试命令（可用于 [ ] 语法）。
> 更多信息：<https://manned.org/test>

- 测试文件是否存在：
  `test -e {{文件路径}}`

- 测试文件是否为普通文件：
  `test -f {{文件路径}}`

- 测试文件是否为目录：
  `test -d {{目录路径}}`

- 测试文件是否可执行：
  `test -x {{文件路径}}`

- 测试字符串是否为空：
  `test -z "{{字符串}}"`

- 测试两个字符串是否相等：
  `test "{{字符串1}}" = "{{字符串2}}"`

- 测试两个整数是否相等：
  `test {{整数1}} -eq {{整数2}}`

- 测试整数是否大于：
  `test {{整数1}} -gt {{整数2}}`

- 使用方括号语法（注意空格）：
  `[ -f {{文件路径}} ] && echo "是文件"`

- 组合条件（与、或、非）：
  `test {{条件1}} -a {{条件2}}`
  `test {{条件1}} -o {{条件2}}`
  `test ! {{条件}}`
