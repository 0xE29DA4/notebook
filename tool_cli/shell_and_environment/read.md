# read

> 从标准输入读取一行并将其拆分为字段赋给变量。
> 更多信息：<https://manned.org/read`

- 读取输入到变量：
  `read {{变量名}}`

- 显示提示信息：
  `read -p "{{提示信息}}" {{变量名}}`

- 设置超时时间：
  `read -t {{秒数}} {{变量名}}`

- 读取密码（不显示输入）：
  `read -s {{变量名}}`

- 读取多个变量：
  `read {{变量1}} {{变量2}}`

- 使用默认变量 $REPLY

  ```sh
  read -p "确认吗? "
  echo "你的回答是: $REPLY"
  ```
