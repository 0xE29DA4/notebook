# set

> 设置或显示 shell 变量和选项。
> 更多信息：<https://manned.org/set>

- 显示所有 shell 变量：
  `set`

- 设置变量的值：
  `set {{变量名}}={{值}}`

- 启用选项（如遇到错误立即退出）：
  `set -e`

- 启用调试模式（打印执行的每条命令）：
  `set -x`

- 禁用选项：
  `set +x`

- 启用多个常用选项（错误退出、未定义变量报错、调试）：
  `set -eux`

- 使脚本在引用未定义变量时退出：
  `set -u`

- 将位置参数重新设置：
  `set -- {{参数1}} {{参数2}}`
