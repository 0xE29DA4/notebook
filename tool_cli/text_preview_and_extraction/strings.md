# strings

> 显示文件中的可打印字符字符串。
> 更多信息：<https://manned.org/strings>

- 显示文件中所有可打印字符串：
  `strings {{文件}}`

- 设置最小字符串长度（默认 4）：
  `strings -n {{长度}} {{文件}}`

- 显示字符串在文件中的偏移量：
  `strings -t d {{文件}}`

- 只显示特定编码的字符串：
  `strings -e {{s|l|b}} {{文件}}`
  - `s` - 单字节字符（默认）
  - `l` - 16 位小端
  - `b` - 16 位大端

- 扫描整个文件（不仅是数据段）：
  `strings -a {{文件}}`

- 从二进制文件中提取字符串：
  `strings {{/usr/bin/ls}}`
