# more

> 分页显示文件内容。
> 更多信息：<https://manned.org/more>

- 分页显示文件：
  `more {{文件}}`

- 从指定行开始显示：
  `more +{{行号}} {{文件}}`

- 显示时提示按键帮助：
  `more -d {{文件}}`

- 压缩连续空行为一行：
  `more -s {{文件}}`

- 显示文件时从第一屏开始：
  `more +1 {{文件}}`

- 交互操作（在 more 内）：
  - `空格` - 显示下一屏
  - `回车` - 显示下一行
  - `b` - 显示上一屏
  - `q` - 退出
  - `/模式` - 搜索字符串
  - `h` - 显示帮助
