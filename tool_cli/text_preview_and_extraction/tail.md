# tail

> 输出文件的结尾部分。
> 更多信息：<https://manned.org/tail>

- 显示文件最后 10 行（默认）：
  `tail {{文件}}`

- 显示文件最后 N 行：
  `tail -n {{N}} {{文件}}`

- 实时追踪文件更新（常用于查看日志）：
  `tail -f {{/var/log/syslog}}`

- 从第 N 行开始显示：
  `tail -n +{{N}} {{文件}}`

- 显示多个文件的结尾：
  `tail {{文件1}} {{文件2}}`

- 显示文件最后 N 个字节：
  `tail -c {{N}} {{文件}}`
