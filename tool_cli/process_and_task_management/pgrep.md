# pgrep

> 根据名称和其他属性查找或发送信号给进程。
> 更多信息：<https://manned.org/pgrep>

- 按进程名查找进程：
  `pgrep {{进程名}}`

- 查找进程并显示完整命令行：
  `pgrep -a {{进程名}}`

- 查找进程并显示 PID 和进程名：
  `pgrep -l {{进程名}}`

- 按用户查找进程：
  `pgrep -u {{用户名}} {{进程名}}`

- 按组查找进程：
  `pgrep -G {{组名}} {{进程名}}`

- 按精确进程名匹配：
  `pgrep -x {{精确进程名}}`

- 查找最老的进程：
  `pgrep -o {{进程名}}`

- 查找最新的进程：
  `pgrep -n {{进程名}}`

- 按父进程 ID 查找子进程：
  `pgrep -P {{父进程ID}}`

- 按会话 ID 查找进程：
  `pgrep -s {{会话ID}}`

- 按终端查找进程：
  `pgrep -t {{终端设备}} {{进程名}}`

- 显示进程的详细信息（PID、PPID、用户等）：
  `pgrep -fl {{进程名}}`

- 反向匹配（不包含指定模式的进程）：
  `pgrep -v {{进程名}}`

- 使用正则表达式匹配：
  `pgrep -f "{{正则表达式}}"`

- 限制输出结果数量：
  `pgrep -n {{数量}} {{进程名}}`

- 分隔输出结果（默认换行）：
  `pgrep -d "," {{进程名}}`

- 查找并返回第一个匹配的 PID：
  `pgrep -o {{进程名}} | head -n 1`

- 组合多个条件查找：
  `pgrep -u {{用户名}} -x {{进程名}}`

- 查找运行时间超过指定秒数的进程：
  `pgrep -o {{进程名}} --oldest`
