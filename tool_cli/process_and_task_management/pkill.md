# pkill

> 按名称向进程发送信号。
> 更多信息：<https://manned.org/pkill>

- 终止匹配名称的所有进程：
  `pkill {{进程名}}`

- 强制终止进程：
  `pkill -9 {{进程名}}`

- 按用户终止进程：
  `pkill -u {{用户名}} {{进程名}}`

- 发送指定信号：
  `pkill -{{信号}} {{进程名}}`

- 按完整命令行匹配：
  `pkill -f "{{命令行模式}}"`

- 终止最近启动的进程：
  `pkill -n {{进程名}}`

- 显示将被终止的进程（不实际终止）：
  `pkill -l {{进程名}}`
