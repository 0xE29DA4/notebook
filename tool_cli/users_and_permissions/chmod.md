# chmod

> 改变文件模式（权限）。
> 更多信息：<https://manned.org/chmod>

- 设置所有者读写执行，组和其他人读执行（755）：
  `chmod 755 {{文件}}`

- 为文件添加可执行权限：
  `chmod +x {{文件}}`

- 递归修改目录权限：
  `chmod -R 755 {{目录}}`

- 为所有用户添加写权限：
  `chmod a+w {{文件}}`

- 移除组和其他人的写权限：
  `chmod go-w {{文件}}`

- 设置所有者读写，组和其他人只读（644）：
  `chmod 644 {{文件}}`

- 显示修改过程：
  `chmod -v 755 {{文件}}`
