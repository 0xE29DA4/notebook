# xxd

> 将文件转换为十六进制、二进制或八进制格式，也可以反向转换。
> 更多信息：<https://manned.org/xxd>

- 以十六进制格式显示文件：
  `xxd {{文件}}`

- 显示文件的前几个字节：
  `xxd -l {{字节数}} {{文件}}`

- 按指定偏移量开始显示：
  `xxd -s {{偏移量}} {{文件}}`

- 显示文件的特定范围：
  `xxd -s {{起始偏移}} -l {{长度}} {{文件}}`

- 以二进制格式显示：
  `xxd -b {{文件}}`

- 以八进制格式显示：
  `xxd -o {{文件}}`

- 禁用十六进制偏移量显示：
  `xxd -p {{文件}}`

- 按指定字节数分组显示：
  `xxd -g {{字节数}} {{文件}}`

- 显示每行的字节数：
  `xxd -c {{每行字节数}} {{文件}}`

- 反向转换（十六进制到二进制）：
  `xxd -r {{十六进制文件}}`

- 将十六进制文本转换为二进制文件：
  `echo "48656c6c6f" | xxd -r -p > {{输出文件}}`

- 从标准输入读取并转换：
  `cat {{文件}} | xxd`

- 创建十六进制转储文件：
  `xxd {{输入文件}} > {{输出文件.hex}}`

- 比较两个文件的十六进制差异：
  `diff <(xxd {{文件1}}) <(xxd {{文件2}})`

- 查看可执行文件的文件头：
  `xxd -l 64 {{可执行文件}}`

- 查看文件的 ASCII 部分：
  `xxd {{文件}} | grep -v "^000000"`

- 转换字符串为十六进制：
  `echo -n "{{字符串}}" | xxd -p`

- 转换十六进制为字符串：
  `echo "{{十六进制}}" | xxd -r -p`

- 查看网络数据包（配合 tcpdump）：
  `tcpdump -x -c {{包数}} {{过滤条件}} | xxd`

- 编辑文件的十六进制内容：
  `xxd {{文件}} > {{临时文件}}`
  `# 编辑 {{临时文件}}`
  `xxd -r {{临时文件}} > {{文件}}`

- 查看图片文件的十六进制头部信息：
  `xxd -l 32 {{图片文件}}`

- 查看压缩文件的魔数：
  `xxd -l 4 {{压缩文件}}`

- 生成指定大小的十六进制数据：
  `dd if=/dev/zero bs={{大小}} count=1 | xxd -l {{长度}}
