# zip

> 打包和压缩文件（.zip 格式）。
> 更多信息：<https://manned.org/zip>

- 压缩目录：
  `zip -r {{归档.zip}} {{目录}}`

- 压缩多个文件：
  `zip {{归档.zip}} {{文件1}} {{文件2}}`

- 添加文件到现有归档：
  `zip {{归档.zip}} {{新文件}}`

- 压缩时排除指定文件：
  `zip -r {{归档.zip}} {{目录}} -x "{{*.log}}"`

- 设置压缩级别（0-9，9 最高）：
  `zip -{{9}} -r {{归档.zip}} {{目录}}`

- 加密压缩：
  `zip -e -r {{归档.zip}} {{目录}}`

- 显示压缩进度：
  `zip -r {{归档.zip}} {{目录}} -v`
