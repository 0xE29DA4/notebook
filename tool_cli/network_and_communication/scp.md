# scp

> 通过 SSH 在主机间安全复制文件。
> 更多信息：<https://manned.org/scp>

- 复制文件到远程主机：
  `scp {{本地文件}} {{用户@主机:远程路径}}`

- 从远程主机复制文件：
  `scp {{用户@主机:远程文件路径}} {{本地路径}}`

- 递归复制目录：
  `scp -r {{本地目录}} {{用户@主机:远程路径}}`

- 指定端口：
  `scp -P {{端口}} {{文件}} {{用户@主机:路径}}`

- 显示复制进度：
  `scp -v {{文件}} {{用户@主机:路径}}`

- 使用压缩传输：
  `scp -C {{文件}} {{用户@主机:路径}}`

- 在两台远程主机间复制：
  `scp {{用户1@主机1:文件路径}} {{用户2@主机2:目标路径}}`
